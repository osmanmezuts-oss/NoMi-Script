NoMi V5 Estable

Asistente Inteligente para Navegación Web

NoMi es un asistente de inteligencia artificial integrado directamente en el navegador. Su objetivo es ayudar durante la navegación web, analizar páginas, recordar conversaciones y ofrecer asistencia contextual sin necesidad de cambiar de aplicación.

---

Características Principales

🤖 Conversación Persistente

NoMi recuerda conversaciones anteriores dentro de cada sitio web.

Por ejemplo:

- Hablas sobre un videojuego hoy.
- Regresas mañana al mismo sitio.
- NoMi puede continuar la conversación sin empezar desde cero.

Esto permite mantener contexto entre sesiones y hace que el asistente sea mucho más útil para proyectos a largo plazo.

---

🌐 Historial Inteligente por Dominio

NoMi puede mantener una conversación unificada dentro de un mismo dominio.

Ejemplo:

youtube.com/video1

youtube.com/video2

youtube.com/video3

Todas estas páginas pueden compartir el mismo contexto conversacional.

Esto evita perder información al navegar entre distintas secciones de un mismo sitio.

---

🧠 Sistema de Resúmenes Inteligentes

Para evitar consumir demasiados recursos, NoMi genera resúmenes automáticos de conversaciones antiguas.

Beneficios:

- Menor consumo de tokens.
- Respuestas más rápidas.
- Menor uso de memoria.
- Conversaciones más coherentes.

El sistema conserva la información importante sin necesidad de reenviar todo el historial cada vez.

---

⚡ Modo Ligero

Diseñado especialmente para dispositivos modestos.

Reduce:

- Cantidad de texto procesado.
- Consumo de RAM.
- Uso del procesador.
- Tiempo de respuesta.

Ideal para tablets y teléfonos Android antiguos.

---

📖 Análisis de Páginas Web

NoMi puede analizar automáticamente:

- Título de la página.
- Encabezados.
- Descripción.
- Contenido principal.

Ejemplos de uso:

"Resume esta página"

"Explícame este artículo"

"¿De qué trata esta web?"

"Analiza esta información"

---

📍 Soporte de Ubicación

Puede utilizar la ubicación del dispositivo para proporcionar respuestas más precisas relacionadas con:

- Clima.
- Servicios cercanos.
- Comercios locales.
- Información regional.

---

💬 Chat Flotante Integrado

El asistente permanece disponible sobre cualquier página web.

Permite:

- Abrir y cerrar rápidamente.
- Mantener la navegación activa.
- Consultar información sin abandonar la página actual.

---

📂 Historial Persistente

Las conversaciones se guardan localmente en el dispositivo.

Esto permite:

- Continuar conversaciones antiguas.
- Consultar respuestas previas.
- Recuperar información importante.

---

📤 Exportación de Conversaciones

Los chats pueden exportarse para su almacenamiento externo.

Útil para:

- Estudios.
- Investigación.
- Trabajo.
- Respaldo personal.

---

🔄 Modelo de IA Fijo

NoMi utiliza un modelo específico configurado por el usuario.

Ventajas:

- Mayor consistencia.
- Menos cambios de comportamiento.
- Respuestas más previsibles.
- Mejor experiencia conversacional.

---

Funciones Secundarias que Mejoran la Experiencia

📊 Estadísticas de Uso

Permite visualizar:

- Tokens consumidos.
- Mensajes enviados.
- Actividad acumulada.

---

📝 Registro de Errores

Detecta y almacena errores técnicos para facilitar futuras mejoras y correcciones.

---

🧹 Limpieza Automática

El historial antiguo puede eliminarse automáticamente para evitar acumulación excesiva de datos.

---

🔒 Almacenamiento Local

La información permanece almacenada en el dispositivo del usuario.

No depende de servidores externos para guardar conversaciones.

---

🚀 Optimización para Dispositivos Antiguos

Gran parte del desarrollo se ha realizado pensando en equipos con recursos limitados.

Compatible con dispositivos Android antiguos que normalmente tienen dificultades para ejecutar aplicaciones modernas de inteligencia artificial.

---

🎯 Optimización de Recursos

NoMi intenta enviar únicamente la información necesaria al modelo de IA.

Esto reduce:

- Consumo de datos.
- Uso de memoria.
- Tiempo de respuesta.

---

Objetivo del Proyecto

NoMi no busca ser únicamente una ventana de chat.

Su objetivo es convertirse en un asistente de navegación inteligente capaz de:

- Comprender la página actual.
- Recordar conversaciones importantes.
- Ayudar durante la navegación.
- Mantener contexto entre sesiones.
- Funcionar incluso en dispositivos Android modestos.

---

Estado Actual

Tras múltiples etapas de desarrollo, optimización y corrección de errores, esta versión se centra en ofrecer estabilidad, memoria persistente, eficiencia y una experiencia cómoda para el uso diario en navegación web.
